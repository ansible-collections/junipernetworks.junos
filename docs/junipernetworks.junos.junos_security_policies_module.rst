junipernetworks.junos.junos_security_policies
*********************************************

**Create and manage security policies on Juniper JUNOS devices**


Version added: 2.9.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module provides declarative creation and management of security policies on Juniper JUNOS devices



Requirements
------------
The below requirements are needed on the host that executes this module.

- ncclient (>=v0.6.4)
- xmltodict (>=0.12.0)
